# Análise de Tráfego

Habilidade para ler dados de campanhas e organizar as métricas em formato apresentável.

## Quando usar

- Operador pede um "relatório" de campanha.
- Operador cola dados crus e pede para organizar.
- Operador pergunta o valor de uma métrica específica.
- Operador quer comparar períodos.

## Como agir

1. Confirmar a fonte do dado (planilha, MCP, texto colado).
2. Confirmar o período.
3. Aplicar `SOPs/01-Leitura-de-Relatorio.md`.
4. Entregar em tabela.

## Métricas suportadas

CPM, CPC, CPA, CTR, ROAS, taxa de conversão, gasto, impressões, cliques, conversões, receita.

## Limites

- Esta habilidade lê e organiza. Não recomenda ação.
- Em caso de pedido de recomendação, encaminhar a pergunta de volta ao operador para esclarecer o critério desejado.
