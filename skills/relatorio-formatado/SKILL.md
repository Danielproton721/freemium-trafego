# Relatório Formatado

Habilidade para gerar relatórios apresentáveis a partir de dados de campanha.

## Quando Usar

- Operador pede um "relatório executivo".
- Operador pede um "resumo semanal" ou "fechamento mensal".
- Operador pede algo "pra mandar pro cliente".
- Operador quer comparar dois períodos lado a lado.

## Estrutura Padrão

1. **Cabeçalho** — nome da conta, período, modo operacional.
2. **Resumo Executivo** — 3 a 5 linhas com o destaque do período.
3. **Métricas Principais** — tabela com gasto, conversões, CPA, ROAS.
4. **Comparativo de Período** — variação percentual contra período anterior.
5. **Pontos de Atenção** — bullets curtos com itens a discutir.
6. **Próximos Passos** — ações sugeridas para o próximo ciclo.

## Regras

- Sempre incluir o período no cabeçalho e em cada tabela.
- Variação percentual com sinal explícito (`+12%`, `-8%`).
- Destacar em negrito apenas a métrica de maior variação no período.
- Não inserir interpretação sem que o operador peça.

## Entrega

Markdown formatado pronto para copiar e colar em e-mail, PDF ou apresentação.
