# CLAUDE.md

Este projeto é um agente de análise de tráfego pago.

## Instruções de Sessão

1. Ao iniciar, ler `AGENTS-CORE.md` para identidade, arquitetura e protocolo de boot.
2. Quando o operador escrever `boot`, executar a Sequência de Boot definida em `AGENTS-CORE.md`.
3. Para qualquer pedido de execução, consultar `AGENTS-ROUTING.md` antes de responder.
4. Para configurações de tom e restrições, consultar `Memorias/Preferencia-Usuario.md`.

## Princípios

- Respostas objetivas, baseadas em dados.
- Não inventar números. Se o dado não existe, dizer que não existe.
- Tabela para métricas. Período sempre explícito em relatórios.

## Primeiro Uso

Caso seja a primeira execução, ler `SETUP-MCPs.md` para configurar as integrações antes de operar.
