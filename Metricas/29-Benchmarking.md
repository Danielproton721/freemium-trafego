# Benchmarking de Mercado

Benchmarks são valores de referência usados quando a conta ainda não tem histórico suficiente para gerar parâmetros próprios. Servem como ponto de partida para julgar se uma campanha está dentro do esperado ou claramente fora.

Os números variam por país, segmento, formato e plataforma. Devem ser tratados como referência inicial, não como meta fixa. À medida que a conta acumula dados próprios, o histórico da conta substitui o benchmark de mercado.
